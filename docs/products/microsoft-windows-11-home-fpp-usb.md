## 상품 개요
마이크로소프트 Windows 11 Home FPP USB는 조립 PC를 새로 구매하거나 운영체제가 설치되지 않은 PC를 처음 세팅하는 사용자에게 안정적인 윈도우 11 환경을 제공하는 정품 운영체제입니다. FPP(Full Package Product) 라이선스 방식으로, PC 교체 시에도 기존 라이선스를 새 PC에 이전하여 재사용할 수 있는 장점이 있습니다. USB를 통한 직관적이고 간편한 설치가 가능하여 컴퓨터에 익숙하지 않은 초보자도 쉽게 윈도우 11을 설치하고 활성화할 수 있습니다. 2022년 4월 출시 이후 꾸준히 인기를 얻고 있습니다.

## 구매 전 반드시 확인할 사항

*   **이 제품이 맞는 소비 상황인지**: OS 미포함 PC를 새로 세팅하거나, 기존 PC의 윈도우 버전 교체, 혹은 향후 PC 교체를 고려하여 라이선스 이전이 필요한 경우에 적합합니다.
*   **라이선스 방식 이해**: FPP(Full Package Product)는 PC 변경 시 라이선스를 재사용할 수 있는 장점이 있지만, DSP(Delivery Service Pack)나 ESD(Electronic Software Distribution) 방식보다 초기 구매 비용이 높습니다. 본인의 사용 계획에 맞춰 선택해야 합니다.
*   **PC 최소 시스템 요구 사항**: Windows 11 설치를 위해서는 TPM 2.0, Secure Boot(보안 부팅) 기능이 활성화된 시스템과 최소 4GB RAM, 64GB 저장 공간 등의 사양을 충족해야 합니다. 구형 PC의 경우 호환성을 확인해야 합니다.
*   **가격대 형성**: 정품 라이선스는 정해진 가격대가 있습니다. 지나치게 저렴한 가격으로 판매되는 제품은 기업용 라이선스를 불법적으로 유통하는 경우일 수 있으므로 주의가 필요합니다.
*   **설치 환경 준비**: USB를 통한 설치이므로, 설치 과정 중 전원 공급이 안정적인지 확인하고, 다른 불필요한 USB나 외장 하드는 잠시 제거하는 것이 좋습니다.

*   **USB를 통한 간편하고 직관적인 설치**: 별도의 설치 파일 다운로드 없이 제공된 USB를 연결하고 안내에 따라 진행하면 되어 컴퓨터 초보자도 쉽게 윈도우 11을 설치할 수 있습니다. 한글 버전으로 용어 이해도 용이합니다.
*   **윈도우 11의 깔끔하고 현대적인 사용자 인터페이스(UI)**: 중앙 정렬된 시작 메뉴와 작업 표시줄, 스냅 레이아웃 등 새로운 디자인과 기능이 사용자 경험을 향상시키고 멀티태스킹을 효율적으로 지원합니다.
*   **안정적인 정품 인증 및 보안 업데이트**: 정품 윈도우 사용으로 인증 문구 없이 마음 편하게 사용할 수 있으며, 마이크로소프트의 지속적인 보안 업데이트와 기능 개선을 받을 수 있어 안전합니다.
*   **향상된 성능과 안정성**: 부팅 속도가 빠르고 프로그램 실행이 부드러우며, DirectStorage 및 Auto HDR 기능 지원으로 게임 환경에서도 개선된 성능을 제공합니다.

## 자주 묻는 질문 (FAQ)

Q. 이 제품은 어떤 제품인가요?
A. 마이크로소프트 Windows 11 Home FPP USB는 개인 사용자를 위한 정품 운영체제로, USB를 이용한 간편한 설치가 특징입니다. FPP(처음사용자용) 라이선스 덕분에 PC를 교체하더라도 기존 라이선스를 새 PC에 이전하여 재사용할 수 있어 장기적인 관점에서 효율적입니다.

Q. FPP 라이선스의 가장 큰 장점은 무엇인가요?
A. FPP(Full Package Product) 라이선스는 '처음사용자용'으로, 한 번 구매하면 특정 PC에 귀속되지 않고 여러 번 재설치하거나 PC를 변경할 때 라이선스를 이전하여 계속 사용할 수 있습니다. 이는 조립 PC 사용자나 주기적으로 PC를 교체하는 사용자에게 매우 유리합니다.

Q. Windows 10에서 Windows 11로 무료 업그레이드와 이 제품 구매의 차이점은 무엇인가요?
A. Windows 10에서 11로의 무료 업그레이드는 기존 시스템의 파일을 유지한 채로 업그레이드되는 방식입니다. 반면, Windows 11 Home FPP USB를 구매하여 설치하는 것은 '클린 설치'로, 모든 데이터를 지우고 새롭게 운영체제를 설치하여 시스템이 더 깨끗하고 안정적으로 작동하는 환경을 구축할 수 있습니다. 특히 새 PC에는 클린 설치가 권장됩니다.

Q. Windows 11 Home과 Pro 버전 중 어떤 것을 선택해야 하나요?
A. 일반적인 가정용, 학습용, 개인 업무용으로는 Windows 11 Home 버전으로 충분합니다. Pro 버전은 BitLocker 드라이브 암호화, 로컬 그룹 정책 편집기, 원격 데스크톱 등 비즈니스 및 고급 보안 기능이 추가되지만, 대부분의 개인 사용자에게는 필요하지 않습니다. 따라서 불필요하게 더 비싼 Pro 버전을 구매할 필요는 없습니다.

Q. Windows 11을 설치하려면 TPM 2.0이 반드시 필요한가요?
A. 네, Windows 11의 최소 시스템 요구 사항에는 TPM 2.0(신뢰할 수 있는 플랫폼 모듈)과 Secure Boot(보안 부팅) 활성화가 포함됩니다. 이는 보안 강화를 위한 조치로, 대부분 2017년 이후 출시된 메인보드 및 CPU에서 지원됩니다. 설치 전에 BIOS/UEFI 설정에서 TPM 2.0 및 Secure Boot가 활성화되어 있는지 확인해야 합니다.

Q. 현재 Windows 11의 최신 버전은 무엇인가요?
A. 현재 Windows 11의 최신 주요 버전은 23H2(2023년 10월 출시)이며, 마이크로소프트는 지속적인 업데이트를 통해 기능 개선 및 보안 강화를 제공합니다. 최신 릴리스 정보는 [마이크로소프트 공식 웹사이트](https://learn.microsoft.com/ko-kr/windows/release-health/release-information)에서 확인할 수 있습니다.
